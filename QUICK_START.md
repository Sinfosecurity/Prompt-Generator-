# Quick Start Guide - PromptFoundry

**For Non-Technical Users**

## What is This?

PromptFoundry helps you create clear, detailed instructions (prompts) that AI tools can use to build web applications for you. Think of it as a smart form that turns your idea into something AI can actually build.

## What You Need

- A web browser (Chrome, Safari, Firefox, or Edge)
- An idea for a web app you want to build
- 5-10 minutes to answer questions

## How to Use

### Step 1: Open the App

1. Double-click the `index.html` file in the "Prompt Generator" folder
2. It will open in your web browser
3. You'll see "PromptFoundry" at the top

### Step 2: Answer 6 Questions

The app will guide you through 6 simple steps:

#### Question 1: What type of app?
Choose from options like:
- Invoice generator
- Calculator
- Content writer
- Analyzer
- Or describe your own

**Example:** If you want to create invoices, pick "Invoice generator"

#### Question 2: Who will use it?
Select your target users:
- Beginners
- Students
- Freelancers
- Business owners
- etc.

**Example:** If you're a freelancer building for other freelancers, pick that

#### Question 3: What problem does it solve?
Write 1-2 sentences explaining the pain point.

**Example:** "Freelancers waste hours creating professional invoices and often make calculation mistakes."

**Tip:** Be specific! The clearer you are, the better the result.

#### Question 4: What will users input?
Select what users need to provide (check all that apply):
- Text
- Numbers
- Files
- Images
- etc.

**Example:** For an invoice app: text (client name), numbers (amounts), form fields (date, items)

#### Question 5: What should the app output?
Select what results should show (check all that apply):
- Generated text
- Scores or ratings
- Explanations
- Charts
- Action steps
- etc.

**Example:** For an invoice app: generated invoice (text/PDF), summary, next steps

#### Question 6: How should it look?
Describe the design:
- **Color scheme**: "Blue and white" or "Dark mode" or "Pastel colors"
- **Visual style**: Minimal, Clean, Modern, Playful, Professional
- **Tone**: Professional, Friendly, Neutral, Conversational
- **Platform**: Desktop, Mobile, or Both

**Example:** Blue and white colors, clean style, professional tone, works on both desktop and mobile

### Step 3: Generate Your Prompt

1. Click the green "‚ú® Generate my AI prompt" button
2. Wait 1-2 seconds
3. You'll see your complete prompt appear!

### Step 4: Choose Your AI Builder

At the top, you'll see buttons:
- **Universal** (works with any AI tool)
- **Cursor** (optimized for Cursor)
- **Antigravity** (optimized for Antigravity)
- **v0** (optimized for v0)

Click the one you plan to use. The prompt will adjust automatically.

### Step 5: Copy Your Prompt

Click one of these buttons:
- **üìã Copy as plain text** - Most common
- **üìù Copy as Markdown** - For formatted text

You'll see "‚úì Copied to clipboard!" when it works.

### Step 6: Use Your Prompt

1. Open your AI builder (Cursor, Antigravity, v0, or similar)
2. Paste the prompt (Ctrl+V or Cmd+V)
3. Let the AI build your app!
4. The AI should NOT ask follow-up questions - your prompt has everything it needs

## Troubleshooting

### "Nothing happens when I click Generate"
- Make sure you answered ALL questions in the current step
- Check for red error messages
- Try refreshing the page and starting over

### "Copy doesn't work"
- Your browser might be blocking clipboard access
- Try: manually select all text (Ctrl+A/Cmd+A) and copy (Ctrl+C/Cmd+C)

### "The app won't open"
- Make sure you're double-clicking `index.html`
- If it opens in a text editor, right-click ‚Üí Open With ‚Üí Your Web Browser

## Need Help?

This is a simple tool with no backend. If something isn't working:
1. Refresh the page (F5 or Cmd+R)
2. Try a different web browser
3. Make sure you're using a modern browser (updated in the last year)
